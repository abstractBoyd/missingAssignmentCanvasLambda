# missingAssignmentCanvasLambda
A single-file lambda that is a web service to get all the missing assignments for one or more students using Canvas

The file, once deployed to an AWS Lambda instance, and set up to have a Function URL, serves up HTML+JS as a frontend and has a REST backend.

The Lambda needs these three environment variables defined in the configuration tab:
CANVAS_BASE_URL = https://your=school=specific=host.instructure.com
CANVAS_TOKEN = 57???????your-personal-canvas-token???????????Za
CUTOFF_DATE = 2026-01-25T04:59:59Z

the CANVAS_BASE_URL will be specific to your student's school
The CANVAS_TOKEN will need to be generated by a parent with access to canvas.
The CUTOFF_DATE is necessary if your student has a class that spans multiple semesters, and they didn't get some assignments done, and will never get them done.  Anything before the cutoff date is ignored.

The CANVAS_TOKEN can be generated from the parent Canvas page:
 1. log in
 2. go to "Account" on the left
 3. Select "Settings" (or go to https://your-school-specific-host.instructure.com/profile/settings)
 4. Scroll down to "Approved Integrations"
 5. Click the "+ New Access Token" button
 6. Generate the token
 7. set the expiration date sufficiently in the future, or never expire
 8. copy/save it somewhere
 9. add it to the environment for the lambda
